    miniLA指令集

      芯片和处理器设计是制约我国科学技术进一步发展的“卡脖子”技术之一。尽管我国在军事、超级计算机和航空航天等关键领域已经逐步实现芯片和处理器的自主研发和设计，但与国外相比，我国在芯片和处理器设计以及软件生态建设等方面仍然存在较大短板。

      为了进一步实现芯片和处理器技术的自主可控，龙芯中科基于二十多年的CPU研制和生态建设的积累，于2020年推出了LoongArch指令集架构。该架构包括基础架构和向量指令、虚拟化、二进制翻译等扩展部分，具备良好的自主性、先进性和兼容性。

      miniLA是龙芯LA32R (LoongArch32 Reduced) 指令集的子集，共包含32个通用寄存器和38条指令 (含3R型10条、2RI5型3条、2RI12型14条、1RI20型2条、2RI16型7条和I26型2条)。

    1. miniLA通用寄存器

      miniLA具有32个32位通用寄存器，与LA32R完全相同，如表2-1所示。

    表2-1 miniLA通用寄存器

    寄存器 助记符 释义
    r0 zero 常数0，永远只返回0 (硬件0，只读不可写)
    r1 ra 存放调用子程序(函数)时的返回地址 (Return Address)
    r2 tp 线程指针。多线程应用程序通过该寄存器来访问某个线程的本地数据 (Thread Pointer)
    r3 sp 堆栈指针。对它的调整必须显式地通过指令来实现，硬件不支持堆栈指针的调整 (Stack Pointer)
    r4~r5 a0~a1 存放子程序(函数)调用时的整型参数或返回值
    r6~r11 a2~a7 存放子程序(函数)调用时的整型参数
    r12~r20 t0~t8 存放临时变量，子程序(函数)使用时不自动保存这些寄存器的值，因此调用后它们的值可能被破坏
    r21 u0 其他通用寄存器
    r22 fp / s9 子程序用该寄存器做堆栈帧指针。子程序(函数)必须在返回前恢复这些寄存器的值以保证其没有变化 (Frame Pointer)。或作为静态寄存器
    r23~r31 s0~s8 静态寄存器

      在miniLA架构中，指令集与通用寄存器存在正交关系，即所有的32个通用寄存器都可以被任意指令使用。

      另外，在汇编语言中，既可以通过寄存器号也可以通过助记符来引用一个寄存器。例如，汇编中的“r0”和“zero”均表示第0个寄存器；“r3”和“sp”均表示第3个寄存器。

    注意

      在本课程提供的LA32R汇编器中，使用寄存器时需添加$符号，如$r0、$zero等，否则将报错。

      miniLA指令的格式、功能、语法及解释汇总如表2-2A所示。

    表2-2 miniLA指令总表

      下面对每条指令的格式、功能等给出详细解释，所使用的符号约定如表2-3所示。

    表2-3 符号释义

    符号 释义
    rj、rk、rd 32位通用寄存器的编号 (rj、rk存放源操作数，rd存放目的操作数)
    ui5 5位无符号立即数
    si12 12位有符号立即数
    si20 20位有符号立即数
    offset 指令所使用的立即数 (扩展后)
    offsXX 指令码中的立即数 (XX是表示位宽的数字；扩展前)
    PC 赋值符号右侧的PC表示当前指令的地址
    (rj) 寄存器rj的值 (其他同理)
    Mem[addr] 地址为addr的存储单元的值
    sext(a) a的符号扩展
    zext(a) a的零扩展

    3. 指令详解

    3.1 3R型指令

    add.w

    • 指令名：加法
    • 使用语法：add.w rd, rj, rk
    • 功能描述：把寄存器rj加上寄存器rk，结果写入寄存器rd。忽略算术溢出。
    • 汇编例子： 
      add.w r4, r2, r3    # (x4) ← (x2) + (x3), (PC) ← (PC) + 4

    sub.w

    • 指令名：减法
    • 使用语法：sub.w rd, rj, rk
    • 功能描述：从寄存器rj减去寄存器rk，结果写入寄存器rd。忽略算术溢出。
    • 汇编例子： 
      sub r7, r5, r9    # (r7) ← (r5) - (r9), (PC) ← (PC) + 4

    and

    • 指令名：逻辑与
    • 使用语法：and rd, rj, rk
    • 功能描述：将寄存器rj和寄存器rk进行按位与，结果写入寄存器rd。
    • 汇编例子： 
      and r6, r1, r0    # (r6) ← (r1) & (r0), (PC) ← (PC) + 4

    or

    • 指令名：逻辑或
    • 使用语法：or rd, rj, rk
    • 功能描述：将寄存器rj和寄存器rk进行按位或，结果写入寄存器rd。
    • 汇编例子： 
      or r12, r13, r1    # (r12) ← (r13) | (r1), (PC) ← (PC) + 4

    xor

    • 指令名：逻辑异或
    • 使用语法：xor rd, rj, rk
    • 功能描述：将寄存器rj和寄存器rk进行按位异或，结果写入寄存器rd。
    • 汇编例子： 
      xor r3, r3, r3    # (r3) ← (r3) ^ (r3), (PC) ← (PC) + 4

    sll.w

    • 指令名: 逻辑左移
    • 使用语法：sll.w rd, rj, rk
    • 功能描述：把寄存器rj左移rk位，低位补0，结果写入寄存器rd。rk[4:0]表示左移位数，高位忽略。
    • 汇编例子： 
      sll.w r1, r6, r2    # (r1) ← (r6) << (r2)[4:0], (PC) ← (PC) + 4

    srl.w

    • 指令名: 逻辑右移
    • 使用语法：srl.w rd, rj, rk
    • 功能描述：把寄存器rj右移rk位，高位补0，结果写入寄存器rd。rk[4:0]表示右移位数，高位忽略。
    • 汇编例子： 
      srl.w r5, r3, r4    # (r5) ← (r3) >> (r4)[4:0], (PC) ← (PC) + 4

    sra.w

    • 指令名: 算术右移
    • 使用语法：sra.w rd, rj, rk
    • 功能描述：把寄存器rj右移rk位，高位补rj[31]，结果写入寄存器rd。rk[4:0]表示右移位数，高位忽略。
    • 汇编例子： 
      sra.w r3, r4, r7    # (r3) ← (r4) >> (r7)[4:0], (PC) ← (PC) + 4

    slt (选)

    • 指令名: 小于则置位
    • 使用语法：slt rd, rj, rk
    • 功能描述：比较寄存器rj和寄存器rk，比较时视为有符号数。如果rj更小，向寄存器rd写入1，否则写入0。
    • 汇编例子： 
      slt r1, r2, r0    # (r1) ← ((r2) < (r0)), (PC) ← (PC) + 4

    sltu (选)

    • 指令名: 无符号小于则置位
    • 使用语法：sltu rd, rj, rk
    • 功能描述：比较寄存器rj和寄存器rk，比较时视为无符号数。如果rj更小，向寄存器rd写入1，否则写入0。
    • 汇编例子： 
      sltu r5, r1, r2    # (r5) ← ((r1) < (r2)), (PC) ← (PC) + 4

    3.2 2RI5型指令

    slli.w

    • 指令名：立即数逻辑左移
    • 使用语法：slli.w rd, rj, ui5
    • 功能描述：把寄存器rj左移ui5位，低位补0，结果写入寄存器rd。
    • 汇编例子： 
      slli.w r5, r3, 3    # (r5) ← (r3) << 3, (PC) ← (PC) + 4

    srli.w

    • 指令名：立即数逻辑右移
    • 使用语法：srli.w rd, rj, ui5
    • 功能描述：把寄存器rj右移ui5位，高位补0，结果写入寄存器rd。
    • 汇编例子： 
      srli.w r1, r2, 1    # (r1) ← (r2) >> 1, (PC) ← (PC) + 4

    srai.w

    • 指令名：立即数算术右移
    • 使用语法：srai.w rd, rj, ui5
    • 功能描述：把寄存器rj右移ui5位，高位补rj[31]，结果写入寄存器rd。
    • 汇编例子： 
      srai.w r3, r1, 7    # (r3) ← (r1) >> 7, (PC) ← (PC) + 4

    3.3 2RI12型指令

    addi.w

    • 指令名：立即数加法
    • 使用语法：addi.w rd, rj, si12
    • 功能描述：把符号扩展的立即数加到寄存器rj上，结果写入寄存器rd。忽略算术溢出。
    • 汇编例子： 
      addi.w r3, r3, 4    # (r3) ← (r3) + 4, (PC) ← (PC) + 4

    andi

    • 指令名：立即数逻辑与
    • 使用语法：andi rd, rj, ui12
    • 功能描述：把零扩展的立即数与寄存器rj进行按位与，结果写入寄存器rd。
    • 汇编例子： 
      andi r1, r3, 0xFFE    # (r1) ← (r3) & 0x0000_0FFE, (PC) ← (PC) + 4

    ori

    • 指令名：立即数逻辑或
    • 使用语法：ori rd, rj, ui12
    • 功能描述：把零扩展的立即数与寄存器rj的值进行按位或，结果写入寄存器rd。
    • 汇编例子： 
      ori r6, r2, 0x001    # (r6) ← (r2) | 0x001, (PC) ← (PC) + 4

    xori

    • 指令名：立即数逻辑异或
    • 使用语法：xori rd, rj, ui12
    • 功能描述：把零扩展的立即数与寄存器rj的值进行按位异或，结果写入寄存器rd。
    • 汇编例子： 
      xori r4, r0, 0x002    # (r4) ← (r0) ^ 0x002, (PC) ← (PC) + 4

    slti (选)

    • 指令名: 小于立即数则置位
    • 使用语法：slti rd, rj, si12
    • 功能描述：比较寄存器rj和符号扩展的立即数，比较时视为有符号数。如果rj更小，向寄存器rd写入1，否则写入0。
    • 汇编例子： 
      slti r7, r5, 0    # (r7) ← ((r5) < 0), (PC) ← (PC) + 4

    sltui (选)

    • 指令名: 无符号小于立即数则置位
    • 使用语法：sltui rd, rj, si12
    • 功能描述：比较寄存器rj和符号扩展的立即数，比较时视为无符号数。如果rj更小，向寄存器rd写入1，否则写入0。
    • 汇编例子： 
      sltui r3, r1, -1024    # (r3) ← ((r1) < 0xC00), (PC) ← (PC) + 4

    ld.b (选)

    • 指令名：字节加载
    • 使用语法：ld.b rd, rj, si12
    • 功能描述：从字节地址(rj)+sext(si12)处读取一个字节，经符号扩展后写入寄存器rd。
    • 汇编例子： 
      ld.b r5, r2, -3    # (r5) ← sext(Mem[(r2) - 3][7:0]), (PC) ← (PC) + 4

    ld.bu (选)

    • 指令名：无符号字节加载
    • 使用语法：ld.bu rd, rj, si12
    • 功能描述：从字节地址(rj)+sext(si12)处读取一个字节，并写入寄存器·。
    • 汇编例子： 
      ld.bu r5, r2, -3    # (r5) ← Mem[(r2) - 3][7:0], (PC) ← (PC) + 4

    ld.h (选)

    • 指令名：半字加载
    • 使用语法：ld.h rd, rj, si12
    • 功能描述：从半字地址(rj)+sext(si12)处读取2个字节，经符号扩展后写入寄存器·。
    • 汇编例子： 
      ld.h r5, r2, 1    # (r5) ← sext(Mem[(r2) + 1][15:0]), (PC) ← (PC) + 4

    ld.hu (选)

    • 指令名：无符号半字加载
    • 使用语法：ld.hu rd, rj, si12
    • 功能描述：从半字地址(rj)+sext(si12)处读取2个字节，并写入寄存器rd。
    • 汇编例子： 
      ld.hu r5, r2, 6    # (r5) ← Mem[(r2) + 6][15:0], (PC) ← (PC) + 4

    ld.w

    • 指令名：字加载
    • 使用语法：ld.w rd, rj, si12
    • 功能描述：从字地址(rj)+sext(si12)处读取4个字节，并写入寄存器rd。
    • 汇编例子： 
      ld.w r5, r2, 8    # (r5) ← Mem[(r2) + 8], (PC) ← (PC) + 4

    st.b (选)

    • 指令名：存字节
    • 使用语法：st.b rd, rj, si12
    • 功能描述：将寄存器rd的最低字节存入内存的字节地址(rj)+sext(si12)处。
    • 汇编例子： 
      st.b r9, r11, 5    # Mem[(r11) + 5] ← (r9)[7:0], (PC) ← (PC) + 4

    st.h (选)

    • 指令名：存半字
    • 使用语法：st.h rd, rj, si12
    • 功能描述：将寄存器rd的最低2个字节存入内存的半字地址(rj)+sext(si12)处。
    • 汇编例子： 
      st.h r9, r11, 10    # Mem[(r11) + 10] ← (r9)[15:0], (PC) ← (PC) + 4

    st.w

    • 指令名：存字
    • 使用语法：st.w rd, rj, si12
    • 功能描述：将寄存器rd存入内存的字地址(rj)+sext(si12)处。
    • 汇编例子： 
      st.w r9, r11, 4    # Mem[(r11) + 4] ← (r9), (PC) ← (PC) + 4

    3.4 1RI20型指令

    lu12i.w

    • 指令名：高位立即数加载
    • 使用语法：lu12i.w rd, si20
    • 功能描述：将符号扩展的20位立即数逻辑左移12位，结果写入寄存器rd。
    • 汇编例子：
      lu12i.w r15, 0xABCDE    # (r15) ← 0xABCDE000, (PC) ← (PC) + 4

    pcaddu12i (选)

    • 指令名：PC加立即数
    • 使用语法：pcaddu12i rd, si20
    • 功能描述：把符号扩展的20位立即数逻辑左移12位，再与PC相加，结果写入寄存器rd。
    • 汇编例子：
      pcaddu12i r6, 0x2468A    # (r6) ← (PC) + 0x2468A000, (PC) ← (PC) + 4

    3.5 2RI16型指令

    beq

    • 指令名：相等时分支
    • 使用语法：beq rj, rd, offset
    • 功能描述：若寄存器rj和寄存器rd的值相等，则跳转。跳转的目标地址是当前PC值加上偏移量offset = sext(offs16 << 2)。其中，offs16是指令码中的16位立即数。
    • 汇编例子： 
      beq r0, r1, -4    # PC ← ((r0) == (r1)) ? PC - 4 : PC + 4

    bne

    • 指令名：不相等时分支
    • 使用语法：bne rj, rd, offset
    • 功能描述：若寄存器rj和寄存器rd的值不相等，则跳转。跳转的目标地址是当前PC值加上偏移量offset = sext(offs16 << 2)。其中，offs16是指令码中的16位立即数。
    • 汇编例子： 
      bne r0, r1, 8    # PC ← ((r0) != (r1)) ? PC + 8 : PC + 4

    blt

    • 指令名：小于时分支
    • 使用语法：blt rj, rd, offset
    • 功能描述：若寄存器rj的值小于寄存器rd的值（比较时视为有符号数），则跳转。跳转的目标地址是当前PC值加上偏移量offset = sext(offs16 << 2)。其中，offs16是指令码中的16位立即数。
    • 汇编例子： 
      blt r7, r0, -4    # PC ← ((r7) < (r0)) ? PC - 4 : PC + 4

    bltu (选)

    • 指令名：无符号小于时分支
    • 使用语法：bltu rj, rd, offset
    • 功能描述：若寄存器rj的值小于寄存器rd的值（比较时视为无符号数），则跳转。跳转的目标地址是当前PC值加上偏移量offset = sext(offs16 << 2)。其中，offs16是指令码中的16位立即数。
    • 汇编例子： 
      bltu r7, r0, 8    # PC ← ((r7) < (r0)) ? PC + 8 : PC + 4

    bge

    • 指令名：大于等于时分支
    • 使用语法：bge rj, rd, offset
    • 功能描述：若寄存器rj的值大于等于寄存器rd的值（比较时视为有符号数），则跳转。跳转的目标地址是当前PC值加上偏移量offset = sext(offs16 << 2)。其中，offs16是指令码中的16位立即数。
    • 汇编例子： 
      bge r6, r0, -4    # PC ← ((r6) >= (r0)) ? PC - 4 : PC + 4

    bgeu (选)

    • 指令名：无符号大于等于时分支
    • 使用语法：bgeu rj, rd, offset
    • 功能描述：若寄存器rj的值大于等于寄存器rd的值（比较时视为无符号数），则跳转。跳转的目标地址是当前PC值加上偏移量offset = sext(offs16 << 2)。其中，offs16是指令码中的16位立即数。
    • 汇编例子： 
      bgeu r10, r0, 8    # PC ← ((r10) >= (r0)) ? PC + 8 : PC + 4

    jirl

    • 指令名：无条件跳转并寄存器链接
    • 使用语法：jirl rd, rj, offset
    • 功能描述：将PC+4的值写入寄存器rd，然后无条件跳转。跳转的目标地址是当前PC值加上偏移量offset = sext(offs16 << 2)。其中，offs16是指令码中的16位立即数。
    • 汇编例子： 
      jirl r1, r7, -4    # r1 ← (PC) + 4, (PC) ← ((r7) - 4)

    3.6 I26型指令

    b

    • 指令名：无条件跳转
    • 使用语法：b offset
    • 功能描述：跳转的目标地址是当前PC值加上偏移量offset = sext(offs26 << 2)。其中，offs26是指令码中的26位立即数。
    • 汇编例子： 
      b 0x1234    # (PC) ← (PC) + 0x1234

    bl

    • 指令名：无条件跳转并寄存器链接
    • 使用语法：bl offset
    • 功能描述：将PC+4的值写入寄存器r1，然后无条件跳转。跳转的目标地址是当前PC值加上偏移量offset = sext(offs26 << 2)。其中，offs26是指令码中的26位立即数。bl指令在功能上类似于jirl r1, offs16，但跳转范围大得多。
    • 汇编例子： 
      bl -4    # (r1) ← (PC) + 4, (PC) ← (PC) - 4
